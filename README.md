## ✨ Key Features

- **Dynamic Product Engine:** Seamless product listing with real-time category filtering.
- **Rich Product Experience:** Detailed pages featuring high-quality image galleries and user reviews.
- **Full-Stack Cart Logic:** Integrated **Add to Cart** & **Wishlist** management.
- **Secure Checkout Flow:** Structured address form validation and checkout process.
- **Persistent Order History:** Functional order placement with data saved across sessions.
- **Smart Authentication:** Secure Login/Logout flow with **Protected Routes** (unauthenticated users are automatically redirected).
- **Seamless UX:** Optimized with **Loading Skeletons** and graceful **Error Handling** to prevent app crashes.
- **Adaptive UI:** 100% Responsive design, providing a premium experience on both Mobile and Desktop.
- **State Persistence:** Powered by **localStorage** to ensure your cart survives a page refresh.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS (Neobrutalism Theme)
- **State:** Redux Toolkit
- **Routing:** React Router v6
- **API:** DummyJSON (Centralized API Constants)
- **Icons:** Material Icons & Lucide

## ⚙️ Installation & Setup

Follow these steps to run the project locally on your machine:

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/bkumbhare11/shopmart.git](https://github.com/bkumbhare11/shopmart.git)

   ```

2. **Install Dependency:**

   ```bash
   npm install

   ```

3. **Start Development Server:**
   ```bash
   npm run dev
   ```
